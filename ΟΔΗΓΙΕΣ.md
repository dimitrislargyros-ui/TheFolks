# BarberTemplate — Οδηγίες Setup Νέου Πελάτη

## Βήματα για κάθε νέο κατάστημα

### 1. Firebase (2 λεπτά)
1. Πήγαινε στο https://console.firebase.google.com
2. "Add project" → δώσε όνομα (π.χ. `barbers-giorgos`) → χωρίς Analytics
3. Build → Realtime Database → Create database → **Europe** → **Test mode**
4. ⚙️ Project settings → Your apps → </> (Web) → Register app → αντέγραψε το `firebaseConfig`

### 2. Άνοιξε το `index.html` με Notepad / TextEdit
Βρες αυτό το κομμάτι στην αρχή:
```
const FIREBASE_CONFIG = {
  apiKey:            "PASTE_API_KEY",
  ...
```
Αντικατέστησε τα PASTE_... με τα στοιχεία του βήματος 1.

### 3. (Προαιρετικό) Άλλαξε το εικονίδιο
Αντικατέστησε το `icon.png` με φωτογραφία 180×180px του καταστήματος.

### 4. Deploy στο Netlify
1. Πήγαινε στο https://app.netlify.com/drop
2. Σύρε-άφησε **τον φάκελο** (ΟΛΑ τα αρχεία μαζί)
3. Αντέγραψε το link που σου δίνει

### 5. Πρώτο άνοιγμα από τον πελάτη
- Ο πελάτης ανοίγει το link → βλέπει οθόνη "Καλώς ήρθες"
- Βάζει το όνομα του καταστήματος
- Πατά το κόκκινο εικονίδιο για να ανεβάσει το λογότυπό του
- Έτοιμο!
